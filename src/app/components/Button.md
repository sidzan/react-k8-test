Default Button is primary
```
  <Button>
    Primary
  </Button>
```

Secondary Button
```
  <Button type="secondary">
    Secondary
  </Button>
```

Disabled Button
```
  <Button disabled={true}>
    Disabled
  </Button>
```

Styled Button
```
  const style = {
    border: 0,
    borderRadius: 5,
    backgroundColor: Constants.Color.BLACK,
    color: Constants.Color.BLUE
  };
  <Button style={style}>
    Styled
  </Button>
```
